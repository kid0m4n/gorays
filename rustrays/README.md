# rustrays

This is a Rust port of [the Go port][1] of [the business card raytracer][2].

## Prerequisites

  * rust 0.9

## Usage

    $ rustc --opt-level 3 rustrays/rays.rs
    $ rustrays/rays
    $ open render.ppm


[1]: https://github.com/kidoman/rays/tree/master/gorays
[2]: https://gist.github.com/kid0m4n/6680629
